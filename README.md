## 👩🏻‍💻 프로젝트 소개
이 프로젝트는 Java Spring Boot 3를 기반으로 하여, Maven을 빌드 툴로 사용하여 개발되었습니다. DBMS로는 MySQL을 선택하여, 회사 내의 팀원들이 컴퓨터 사양을 서로 공유할 수 있는 서버 모니터링 시스템을 개발하였습니다.

## ⚒️ 기술 스택
#### Frontend
- HTML/CSS 
- JavaScript (jquery)

#### Backend
- Java (Spring Boot 3)
- Maven (빌드 툴)
- MySQL (DBMS)

## 📖 프로젝트 내용
프로젝트의 핵심 기능은 사내 팀원들이 자신의 컴퓨터 IP와 컴퓨터 이름을 등록하여 사용하고 있는 컴퓨터의 사양을 공유할 수 있도록 하는 것입니다. 
이를 위해 별도의 Agent를 개발하여, 타 사용자 컴퓨터의 정보를 수집하고 서버에 등록할 수 있도록 구현하였습니다.
